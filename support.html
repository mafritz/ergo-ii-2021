Liste de ressources utiles pour le cours.

Slides

Supports utilisés pendant les séances du cours synchrone.

UX pour la conception et évaluation de UI pour le web
Présentation général du domaine UX/UI. La présentation se focalise principalement sur la phase d'implémentation d'une interface concrète. Les techniques de conception et évaluation des phases exploiratoires (e.g. analyse des besoins, persona, ...) ne sont pas abordées.
Prototypage low-fidelity dans l'UX
Présentation sur des techniques de prototypage low-fi dans le UX. Ce type de prototypage s'effectue surtout dans des phases initiales du projet, afin de maximiser l'exploration de variantes en minimisant le travail à accomplir au niveau du design et des fonctionnalités.
Prototypage high-fidelity dans l'UX
Présentation sur des techniques de prototypage high-fi dans le UX. Ce type de prototypage essaie de véhiculer une expérience la plus proche du produit finale au niveau de l'apparence esthétique et/ou des fonctionnalités.
Techniques d'évaluation dans l'UX
Présentation sur des techniques d'évaluation comme l'oculométrie, le test A/B, le test des 5 seconds, les échelles d'utilisabilité/UX, et le test utilisateur.

Logiciels de prototypage

Logiciels pour le prototypage à différents niveaux d'abstraction.

Figma
Logiciel pour le prototypage et les maquettes interactives qui permet de collaborer en temps réel. L'application web marche dans le navigateur ou peut être téléchargée en version desktop. Le plan gratuit permet une collaboration limitée.
Uizard
Logiciel collaborative pour le prototypage rapide. L'application permet d'importer des prototypes papier et les transforme directement en dessin dans le logiciel.
Balsamiq
Logiciel pour le prototypage surtout low-fidelity. Le logiciel peut être utilisé dans le cloud ou en modalité application desktop.
Adobe XD
Logiciel de prototypage de la suite Adobe, il s'intègre avec les autres logiciels de la suite (e.g. Illustrator, Photoshop, etc.).
