Portfolio des étudiant-es

Cette page propose les projets des étudiant-es qui ont accepté de rendre public leurs travaux. Les consignes pour les travaux sont disponibles dans la page des livrables.

Conception d'une maquette interactive avec test utilisateur

Le projet principal du cours consiste dans un travail en binôme divisé en trois étapes : (i) conception d'une première version d'une maquette interactive suite à la récolte d'information pendant le cours ERGO I (e.g., analyse des besoins, trie de cartes, ...) ; (ii) planification, passation et analyse de résultat d'un test utilisateur ; et (iii) conception d'une deuxième version de la maquette interactive suite aux retours du test utilisateur.

Prototype mobile

Les étudiant-es ont créé un prototype mobile sur un sujet de leur choix, en low-fi ou high-fi et avec la technique de leur préférence. Ils ont ensuite enregistré l'utilisation du prototype pour montrer les fonctionnalités interactives.

Test A/B avec données fictives

Les étudiant-es ont proposé un protocole de passation pour un test A/B avec deux variantes d'un site ou application de leur choix et une mesure d'intérêt pour déterminer la meilleure version à garder. Sur la base de la proposition, les étudiant-es ont obtenu un jeu de données fictif, afin d'effectuer un test statistique pour départager les deux versions.
